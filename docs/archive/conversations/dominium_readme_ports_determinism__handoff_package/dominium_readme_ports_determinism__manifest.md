## Recommended Storage

Keep each old-chat package in its own folder. Use the chat label in filenames. Preserve the ZIP if available. Keep Markdown and YAML together. Do not edit the YAML manually unless necessary; if it must be edited, preserve IDs and labels. When combining old chats later, ingest the YAML and aggregator packet first, then use the full report for detail and audit trail.
