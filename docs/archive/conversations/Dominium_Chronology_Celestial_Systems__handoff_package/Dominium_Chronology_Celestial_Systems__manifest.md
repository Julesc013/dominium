## Recommended Storage

- Keep each retired-chat package in its own folder.
- Use the chat label in filenames.
- Preserve the ZIP archive if available.
- Keep Markdown and YAML together.
- Do not manually edit the YAML unless necessary.
- If edits are made, preserve the original ZIP separately.
- When aggregating, ingest the YAML and aggregator packet first, then consult the full report for nuance.

## Final Status

The package is complete and usable with caveats. Main caveats: actual Plan G/The Game architecture was not visible in this chat; scientific constants and celestial lists require verification; some assistant-proposed non-Earth calendar names and leap rules are tentative.
