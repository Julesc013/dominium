## Recommended Storage

Keep each chat package in its own folder. Use the chat label in filenames. Preserve the ZIP if available. Keep Markdown and YAML together. Do not edit the YAML manually unless necessary; if manually edited, record the edit date and reason. When later aggregating, feed at least the full report, spec sheet, aggregator packet, and manifest to the aggregator.
