# Reader Brief — Dominium Robotic Seed-Civilisation Architecture

## What this chat was about

This chat developed Dominium from an enormous deterministic full-scale solar-system/MMO feasibility question into a candidate game identity: a robotic seed-civilisation game. Players are robot bodies fabricated by a mothership that landed on a procedurally generated, science-bounded planet. They use ship knowledge, limited materials, nanobot construction, automation, and machine process graphs to build industry, remote spawn labs, cities, and eventually terraforming/megaproject systems.

## Top 20 things to know

1. The user wants science-bounded procedural solar-system/planet generation.
2. The initial scope is better understood as a star/solar system, not a full galaxy.
3. Planet generation should not rely primarily on manual painting.
4. Painted/data overrides are still needed.
5. A shared field-layer model should support generation, saves, packs, and terraforming.
6. Humans sent a robotic mothership to restart civilisation.
7. Players spawn as robot bodies fabricated by physical machines.
8. Remote spawn labs are a major exploration/expansion milestone.
9. The mothership contains recipes/knowledge so players do not reinvent all technology.
10. The mothership must be resource/throughput/heat/scale limited.
11. Robots provide diegetic HUDs, strength, bodies, tools, and respawn logic.
12. Nanobots provide diegetic blueprint construction and cut/fill manipulation.
13. Nanobots must not be free magic.
14. Worker NPC labour was strongly deprioritised.
15. Automation should exist from the start.
16. Machines/factories should compile into process graphs/equations.
17. Cities should be easy to start but advanced tech should require city-scale industry.
18. Fog of war and sensing limits matter for both gameplay and networking.
19. Unreal is recommended as renderer/client/tooling, not authoritative universe core, pending verification.
20. The best next step is a first playable slice spec from mothership spawn to remote spawn lab.
