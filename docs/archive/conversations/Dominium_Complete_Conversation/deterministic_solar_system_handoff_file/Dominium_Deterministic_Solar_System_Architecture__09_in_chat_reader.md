## Plain-English explanation

The chat’s substance is that Dominium should be shaped as a deterministic robotic seed-civilisation game. The user wants science-bounded procedural planets and solar systems, sparse editable terrain, physical robot spawning, nanobot construction, machine/factory abstraction, fog of war, and possible MMO/private/single-player modes. The strongest design move was replacing worker NPC labour with automation because the civilisation is robotic.
