## Recommended Storage

- Preserve the ZIP if available.
- Keep Markdown and YAML files together.
- Do not edit the YAML manually unless necessary; if edited, record the change externally.
- Later aggregation should ingest the YAML and aggregator packet first, then use the full report for detail.
