# AGENT HISTORY

History MUST be generated automatically from agent actions and process outcomes.
History entries MUST include timestamps and provenance identifiers.
History MUST be deterministic for identical inputs and seeds.
History MUST NOT be treated as objective truth; it may be incomplete or wrong.

History systems MUST NOT assume human authorship, narration, or player curation.
History systems MUST NOT rely on "AI magic" for interpretation.
