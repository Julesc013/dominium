# AGENT CONTRACTS





Contracts MUST be declarative, inspectable data that binds agents and institutions.


Contracts MUST define obligations, allowed processes, and failure clauses.


Contracts MUST be enforceable only via processes and MUST fail visibly.


Contracts MUST NOT encode hidden strategies or assume success.





Contracts MUST NOT assume human legal defaults or player intent.


Contracts MUST NOT rely on "AI magic" or implicit enforcement.
