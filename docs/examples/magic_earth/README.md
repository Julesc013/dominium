# Magical Civilization on Earth (Exemplar)

Purpose: demonstrate magic layered on realistic Earth without engine changes.

## Demonstrates
- Magic reality pack coexisting with scientific overlays.
- Mana and fate fields gated by belief artifacts.
- Institutions enforcing magical norms alongside civic science.

## Dependencies
- Pack: `org.dominium.examples.realistic_earth`.
- Pack: `org.dominium.realities.magic`.
- Packs: `org.dominium.worldgen.real.milkyway`, `org.dominium.worldgen.real.sol`,
  `org.dominium.worldgen.real.earth_macro`.

## Invariants Proved
- Realism and magic are content, not engine law.
- Multiple realities coexist with explicit precedence.
- Subjective gating never mutates objective truth.

## Safe Removal or Modification
- Remove `org.dominium.examples.magic_earth` to disable magic overlays.
- Removing `org.dominium.realities.magic` disables magic capability resolution.
- Keep knowledge artifacts separate from objective data for consistency.
