# Glossary (ARCH0 Binding)

Engine: Mechanisms only.
Game: Meaning and rules.
Module: A composable system with declared invariants.
System: A running instance of a module.
Law: A rule governing existence or action.
Capability: An additive permission.
Policy: A constraint on permitted action.
Event: A scheduled state transition.
Effect: The result of an event.
Derived: Non-authoritative computation.
