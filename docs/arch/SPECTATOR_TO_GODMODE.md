# Spectator to God Mode (OMNI0)

Status: draft.
Scope: authority spectrum as data-defined capability profiles.

## Spectrum Examples
All modes are data-defined profiles using capabilities + denials:
- SPECTATOR_ONLY
- COMPETITIVE_LOCKDOWN
- STANDARD_SURVIVAL
- CREATIVE
- ANARCHY
- META_ANARCHY
- OMNIPOTENT_ADMIN

## Rules
- No mode implies another.
- Denials are explicit and override grants.
- Overrides are explicit effects and audited.
