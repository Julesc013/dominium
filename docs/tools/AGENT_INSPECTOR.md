# AGENT INSPECTOR





Agent inspection MUST expose goals, beliefs, memory, plans, and failures as read-only data.


Agent inspection MUST respect epistemic limits and MUST NOT reveal hidden authority or knowledge.


Agent inspection MUST NOT assume human intent or intelligence.


Agent inspection MUST be deterministic and headless-safe.
