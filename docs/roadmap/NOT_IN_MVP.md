# Not in MVP (ROADMAP)

Status: binding.
Scope: explicit exclusions before SLICE-3 completion. Non-exhaustive.

The following are NOT in MVP and are forbidden before SLICE-3 completion:
- Detailed terrain meshes
- Combat systems
- Narrative or quest engines
- Emotion simulation
- Fluid dynamics solvers
- XP/levels/tech trees
- Interstellar travel
- Multiplayer networking

These items are not forbidden forever. They are forbidden before SLICE-3
completion only.
