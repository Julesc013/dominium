# DECOR/MODEL (Rulepacks)

Decoration authoring-time rulepacks and overrides.

## Responsibilities
- Stable IDs and canonical ordering for rule application.
- Separate authoring data from compiled artifacts.

## Non-responsibilities / prohibited
- No authoritative state; decor is derived/optional.
- No platform/UI/render code.
