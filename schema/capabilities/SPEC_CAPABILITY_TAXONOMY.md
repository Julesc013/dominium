# SPEC CAPABILITY TAXONOMY (OMNI0)

Status: draft.
Version: 1.0
Scope: canonical capability categories and examples.

## Capability Categories
- SIMULATION (mutate authoritative state)
- TEMPORAL (time perception controls)
- SPATIAL (movement and entry)
- EPISTEMIC (information access)
- GOVERNANCE (law/jurisdiction modification)
- EXECUTION (backend/policy controls)
- INTEGRITY (anti-cheat, validation)
- ARCHIVAL (freeze, fork, history)

## Example Capabilities (non-exhaustive)
- CAN_OBSERVE
- CAN_PLAY
- CAN_BUILD
- CAN_TRAVEL
- CAN_MODIFY_LAWS
- CAN_OVERRIDE_CAPS
- CAN_ARCHIVE
- CAN_FORK_TIMELINE

## Rules
- Capabilities are additive.
- Capabilities are scoped by domain and time window.
- Evaluation is deterministic and auditable.
