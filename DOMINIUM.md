# DOMINIUM

Dominium is the product layer that hosts the game, launcher, setup, and tools on
top of the Domino engine.

For the repository overview and invariants, see `README.md`. For authoritative
engineering contracts, see the specs under `docs/`.

- Game product: `docs/SPEC_GAME_PRODUCT.md`

- Setup/installer core: `docs/SPEC_SETUP_CORE.md` and `docs/API/SETUP_CLI.md`
- Tools core: `docs/SPEC_TOOLS_CORE.md`
