# STRUCT/MODEL (Models)

Structure semantic models and authoring-time representations.

## Responsibilities
- Define stable model IDs and canonical storage ordering.
- Keep authoring data separate from compiled runtime artifacts.

## Non-responsibilities / prohibited
- No world-space baked geometry as authoritative state.
- No platform/UI concerns.
