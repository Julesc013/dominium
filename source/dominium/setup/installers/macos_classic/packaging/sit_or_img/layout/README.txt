Classic Mac OS Installer Layout

This directory is the source layout for Classic disk image builds.
Place the following items here before running build_image.sh:

- Installer application (classic_installer_app)
- Manifests/ (DSUM manifests)
- Payloads/ (DSU archives and file sets)
- Docs/ (README, release notes)
- Uninstall/ (optional uninstall tool)

No external downloads or dependencies are required.
