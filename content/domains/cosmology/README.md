# Cosmo Core Data (Authoring)

Cosmos anchor and region definitions live here as authoring sources. They are
compiled to TLV packs by `coredata_compile` and consumed by runtime as
deterministic data.

See:
- `docs/reference/specs/SPEC_CORE_DATA.md`
- `docs/reference/specs/SPEC_COSMO_CORE_DATA.md`
