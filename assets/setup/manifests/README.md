# Setup Manifest Samples

Canonical manifest files (binary TLV):
- `dominium_core.dsumanifest` — minimal runtime-only example
- `dominium_full.dsumanifest` — launcher + runtime + tools + base pack example

Use the dev CLI:
- `dominium-setup manifest-validate --in <file>`
- `dominium-setup manifest-dump --in <file> --out <json>`
